## 📌 Project Overview

**MyPortfolio** is my personal developer portfolio website built to showcase my projects, skills, and experience as an AIML Engineer. The site highlights end-to-end ML pipelines, full-stack projects, and internship work — all presented in a clean, responsive, JavaScript-driven interface.

> 🎯 Built to represent my technical identity as an AI/ML engineer — from model training to deployment.

---

## 🔄 Structure

```
Landing Page → About → Skills → Projects → Experience → Contact
```

### 1️⃣ Hero / Landing Section
- Name, role, and headline introduction
- CTA buttons — Resume link & GitHub / LinkedIn profiles
- Animated entrance with smooth scroll navigation

### 2️⃣ About Me
- Brief background as a B.Tech AIML student at MBIT, CVM University
- Internship experience at Mind Inventory
- Focus areas: ML pipelines, dashboards, and deployment

### 3️⃣ Skills
- Programming: Python, JavaScript
- ML/AI: Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn, Plotly
- Tools: Streamlit, Git, GitHub, Jupyter Notebook
- Web: HTML, CSS, JavaScript

### 4️⃣ Projects Showcase
- **SmartCart** — Customer segmentation dashboard (KMeans + PCA + Streamlit)
- **ShopSmart** — Purchase intent predictor (Decision Tree + GridSearchCV)
- **FoodDelivery** — Full-stack MERN food ordering platform
- **Data Cleaning Python** — Pure Python data cleaning (no libraries)

### 5️⃣ Experience

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/ronakrajput8882/MyPotfolio.git
cd MyPotfolio/Potfolio

# Open directly in browser
open index.html
# or
start index.html   # Windows
```

> No build tools or dependencies required — plain HTML, CSS, and JavaScript.

---

## 🔍 Key Highlights

- ⚡ **Zero dependencies** — pure HTML + CSS + JS, no frameworks needed
- 📱 **Responsive design** — works across desktop, tablet, and mobile
- 🎨 **Smooth animations** — scroll-triggered reveals and hover effects via vanilla JS
- 🔗 **Direct project links** — each project card links to the live GitHub repo
- 📄 **Resume integration** — one-click access to Google Drive hosted resume

---

## 🧠 Key Learnings

- Building interactive UIs without a framework deepens understanding of the DOM and event model
- CSS transitions and JS-driven class toggling can achieve smooth animations without any animation library
- Structuring a single-page portfolio with clear section flow improves visitor engagement
- Hosting static sites is free and instant — no backend required for a personal portfolio
